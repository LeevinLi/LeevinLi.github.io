ADB常用命令

ADB常用命令

其他常用

安装与卸载

adb install -r -d -t xx
adb uninstall xx

查看系统所有属性

adb shell getprop

获取系统属性

adb shell getprop xx

获取CPU架构

adb shell getprop ro.product.cpu.abi

截图

adb shell screencap -p /sdcard

录屏

adb shell screenrecord /sdcard/video.mp4

导出Trace文件

adb pull /data/anr .

导出外部存储数据库

adb pull /data/data/com.android.providers.media/databases/external.db

导出外部存储数据库R

adb pull /data/data/com.google.android.providers.media.module/databases/external.db

PackageManager相关，简写pm

查看设备所有应用包名[-s: 系统应用 -3:三方应用]

adb shell pm list packages -s/3

查询应用安装路径

adb shell pm list packages -f | grep 「应用名称」

删除应用信息根据包名

adb shell pm clear 「包名」

设置应用可用/不可用

adb shell pm enable/disable-user 「包名」

dumpsys相关

查看应用信息

adb shell dumpsys package 「包名」

查看应用内存

adb shell dumpsys meminfo 「包名」

查看JOB任务

adb shell dumpsys jobscheduler

查看设备分辨率

adb shell vm size

查看设备屏幕密度

adb shell vm density

查看当前焦点页面

adb shell dumpsys activity activities | egrep -in –color ‘ResumedActivity’
adb shell dumpsys window | egrep -in –color ‘mCurrentFocus’

查看某应用正在运行的Services，根据包名

adb shell dumpsys activity services 「包名」

ActivityManager相关，简写am

监视Crash 或者ANR错误

adb shell am monitor

强制停止进程根据包名

adb shell am force-stop app 「包名」

启动某页面

adb shell am start -n xx/.xx

启动页面并携带ACTION，mimetype，data信息

adb shell am start -n xx/.xx -a android.intent.action.VIEW -t image/* -d content://media/external/images/media/2017

发起ACTION并携带参数

adb shell am start -a android.intent.action.VIEW -d file:// -t image/* -c android.intent.category.DEFAULT

启动页面并携带自定义参数

adb shell am start -n xx/xx -e key value

Intent参数规范

String 可空String Boolean Int Long Float Uri
参数 -e / -es -esn -ez -ei -el -ef -eu
类型 String String(null) boolean int long float uri

更多可参考官方说明 Android Developer

日志相关

清除日志缓存

adb logcat -c

将日志保存到文件中

adb logcat > logcat.log

将日志保存到文件中并过滤日志级别

adb logcat -s *:E > logcat.log

搜索日志

adb logcat | egrep -in –color ‘xx’

搜索ActivityTaskManager日志

adb shell logcat -b all | egrep -in –color ‘ActivityTaskManager:*’

跳过开机向导

方法1：
存在缺点，会导致系统中部分功能无法使用

adb shell pm disable com.google.android.setupwizard

方法2：
改变设置属性，执行完开机向导的流程其实就是设置了这两个属性值，重启使其生效

adb shell settings put secure user_setup_complete 1
adb shell settings put global device_provisioned 1
adb reboot
