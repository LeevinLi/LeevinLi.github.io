2020年终总结

  回想起2020年，这一年中发生了很多大事，而其中最重要的一件就是我又多了一个身份，完美升级当奶爸了，这其中的喜悦不甚言表，尤其是看着现在五个月大的儿子健康可爱，活泼爱笑，写到这里心中已经浮现出了儿子笑颜，心中不禁升起一片暖阳，当然这一切都离不开我伟大的老婆，是我们一起携手共同一点一滴创建起这个温馨的小家。
  今年春节临近，因为疫情影响加之儿子还小，不宜在舟车劳顿赶春运了，所以从年初就和家人商量着今年不回去过春节了，所以目下也没有了往年每天紧张关心抢票的事情，每周末可以悠闲在家陪陪儿子，备备年货之类的。想想自己马上而立之年还从未在外过过年，自从工作以来每年都是踏准时间点冒着人海往回赶，不论路途是多么曲折，不过随着近几年高铁的高速发展，交通确实便利了不少，现在还深刻的记得15年春节过后坐大巴车来上海，由于天气不好大面积霜冻下雪，一路上走走停停足足走了两天48小时，也创造了旅途曲折的最高纪录。
  回想这一年仿佛在昨夜的梦里一般，从2019-10月开始备孕到2020-01-14日在曲阳医院确认怀孕，一切的一切都按照计划如期而至，得知要当爸爸的那一刻高兴的像个孩子，一时间除了喜悦之外觉的自己又多了一份责任，多了许多有意义的事情需要去一一完成。检查完和父母也分享了喜悦的心情，随之而来的就是各种注意事项的嘱咐。
  没过几天，我们就收拾好行李准备回家过春节了，2020-01-19日怀着急切，愉悦的心情踏上返乡的列车，可在家还没呆几天疫情大爆发了，天天看着新闻感受着全国疫情蔓延，顿时感觉一场前所未有的灾难大片在身边上演，计划好的行程也全部打乱。2月初接到了公司复工的通知，2020-02-03和老婆一到冒着疫情的危险，一路避开人流密集区小心翼翼的来到上海，02-04完成大采购囤积了一批食物后开始了为期一个月的居家办公生活，现在想来那段日子还是蛮惬意的，工作上居家办公自然没有在公司的效率高，最头疼的还是代码的同步问题，日常还是以维护现有项目稳定为主，除此还在那段时间完成了照片压缩功能和应用自动化测试的一部分脚本，生活上更是满足了一个吃货的基本需求，和老婆在家每天倒腾各种美食，饺子，包子，香锅等等，一个月下来肚子上的赘肉又大了一圈。随着疫情的好转，公司也通知分批复工了，记得03-09日第一天回公司上班，马路，地铁，公司，园区相比往常人少了许多，每个人也都是严密防护，上班期间仍需要带上口罩，一天下来鼻梁、耳朵已是疼痛难忍，也正是这样的状态又持续了一个月左右，转眼来到了四月，老婆的肚子也一天天隆起，孕期在悄然之中已经有六个月了，时不时的可以看见肚子里的小生命有动静了，每次体检也都是一路绿灯，相信宝宝在老婆的肚子里正在健康的成长。随之而来的是老婆的饮食问题，由于周内工作都太忙了没时间做饭，平常都是吃外卖，显然这样的方式已经无法让人接受了。04-11日是个下雨天早晨，我去车站接我妈过来专职给我们做饭，以满足孕期的饮食营养和健康。
  转眼夏天已然过去，孕期也到了后期，我和老婆商量还是提前回老家生产，不论是所需的费用还是后期月子的照顾方面都会更好一些，08-01日老婆开始休产假，一早我送她们搭上了回家的高铁，送别的那一刻，我多想再为老婆多做些事情，多想能够陪在她身边，我知道在这个阶段她是需要我的，哪怕像在上海一样白天上班每天下班回去能见到也是好的，可惜这一切并不现实，这或许就是打工人的两难的。我怀着复杂的心情走出车站，坚持每天晚上打视频，主动和老婆打视频聊聊天，我知道她第一次在没有我在的情况下和父母一起生活确实也有些不适应，好在一切都坚持下来了。
  离预产期的日子越来越近，08-22日我请了陪产假从上海回汉中，当天下午就到家了，吃完饭陪老婆在江边散步加锻炼，因为都说孕后期多走路有助于生产，在加上刚回去有说不完的话，当天下午走了很大一圈，走的路程比往常都要多一些，没想到休息了一晚之后第二天一大早发现羊水破了，急忙提上早已准备好的日常用品赶往医院，医生确认羊水破了当即通知需要住院，通知完父母后一大家子都紧张忙碌了起来，都赶到医院等候着家庭新成员的降生。第一天，第二天，在催产针的药力作用下一方面腹痛难忍而宫口并没有达到能够生产的程度，另一方面羊水已经破了，医生告诉说从羊水破算起三天内婴儿必须出来否则影响健康。老婆在多重压力下心理上也显得着急了，都憋着一股劲要在第三天努力把货给卸了，凌晨五点左右腹痛加剧，送进产房后继续注射催产针，直到中午12点多药物、物理方式都尝试过了，还是没达到要求。医生看着老婆情绪有些不稳定，让我进产房陪产，在产房的一个多小时里我不停的和老婆说话分散她的注意力分散疼痛感，14点左右疼痛感已经到了无法忍受的程度，商量着请医生再检查一次宫口情况，如果情况还不是太理想则放弃顺产，好在情况出现了一些转机，发现具备生产条件了，接下来的半小时就是作为母亲这一辈子所经历的最漫长最痛苦的半小时，14点42分，长着一头乌黑浓密头发，6.7斤重的大胖小子出生了，母子平安。当听到宝宝第一声啼哭时激动的不知所措，热泪盈眶，一时间出产房报喜都语无伦次，自此一家人悬着的心终于放下了，整个孕期最凶险的时刻也过去了，而这一天正好是七夕节（08-25日），宝宝的出生也正是我和老婆所期待的最好的七夕礼物。
  由于顺产，第二天就可以出院，住进了离家很近的一家月子中心，父母来回照顾都方便。而我也要忙着给宝宝取名，办出生证、户口等一些事情。本来和老婆商量着我们自己给宝宝起名字的，我们提前也准了许多寓意好的字，最后还是无奈才疏学浅，也不懂的什么五行八卦，还是找了专业的人办专业的事，这样一来大家都安心。乳名我们家庭内部也是取舍半天，最后决定还是采用他姥姥建议的“乐乐”。出生后的一些事情处理完了，半个月的陪产假余额也已不足，我马上又要赶回上海上班了，而这次走心里至少踏实了许多，让我在远在千里之外能够安心的上班。平时都是抽乐乐醒着的时候视频通话，和老婆聊聊天看看儿子的近况，可这种隔着屏幕的想念着实让人有些难受，每每看着儿子成长变化越来越大，表情越来越丰富，脑海里浮现的全是儿子的画面。
  这种思念的日子一直持续了三个月，而这期间除了十一国庆节我回家陪伴了几天之外，大多数时间都是我一个人在上海这边，感觉一下子回归到了单身生活的状态，经常性的选择加班或者在公司上完课在回家，周末也回归到了躺尸运动中，一个人的饭食也懒的做了嫌麻烦，干脆全部买着吃，没事了再打打游戏，也正是那段时间第一次上了王者，可能是运气好吧，感觉上王者也并没有那么难。这样的日子虽然自由却缺少灵魂，日子久了整个人都会意志消沉。好在乐乐很快过100天了，11-27日，母亲和老婆带着乐乐来上海了，我也结束了这自由散漫的日子，那天我早早的下班去车站接他们，我原本还一直担心小乐乐第一次坐高铁会不会不习惯，汉中-西安段隧道很多乐乐会不会耳鸣，车上人太多他会不会害怕，在车上会哭闹之类的，结果证明我所有的担心都是多余的，小乐乐第一次出门见到很多新奇的人和物，高兴的一路都没怎么睡觉。列车到站已经晚上十点了，接到他们之后乐乐还是醒着的，一逗都会笑了，看着眼前这个小家伙，忍不住的想好好保护他，让他快乐的成长。
  自从乐乐过来之后，我们的生活一下子变得忙碌，充实，欢乐起来了。老婆过来忙完生育金，疫苗等事情休整了半个月产假也结束了，也该回公司上班了，一切回归到了正轨，周内白天我们都不在，晚上回去可以逗逗乐乐，哄他睡觉，而有的时候加班回去他已经睡着了，第二天早上走的时候他还依然在呼呼大睡，最珍贵的还是周末，可以多陪陪他，给他喂奶，换尿包，洗澡等等，尽到一个合格奶爸的责任。从11月过来至今已经快三个多月了，小乐乐也快六个月大了，从刚过来的时候只会简单的笑，到现在除了会笑，还会高兴的发出各种声音，听觉也很灵敏，两只小手诈唬的不停，在大人怀里不停的抓来抓去，两只脚也是一样，不停的蹦跶就想弹簧一样，看着他一天天的成长，学会的本领也越来越多，我的心里甚是欣慰。
  2020年注定是不平凡的一年，一场疫情席卷了全世界，各行各业都受到不同程度的影响，现在回望这场疫情已经持续了整整一年仍不见有好转的迹象，真希望这场疫情能早日消失匿迹，早日能回归正常的生活。对于我来说，自己已经到了而立之年，这一年也成功升级做了爸爸，家庭的和谐温馨让我内心更加温暖，而工作和事业方面我却仍然有许多的迷茫，我给自己定的目标是，没有明确方向的时候埋头努力就好了，希望2021年好运能够延续，家庭幸福，工作顺利，过的快乐而充实。

sleep()与wait()区别

  • sleep()简介

    1. sleep()方法用于线程控制，调用后使当前线程进入TIMED_WAITING状态，此状态下会让出CPU的资源并且不释放锁(sleep()与锁没直接关系)
    2. sleep()方法是Thread类中的静态方法，它不能改变对象的机锁，当在一个Synchronized块中调用sleep()方法时，线程虽然休眠了但是对象的机锁并没有被释放，其他线程无法访问这个对象
    3. sleep()休眠时间到达后，当前线程会进入READY状态，等待CUP调度执行

  • wait()简介

    1. wait()用于线程间的通信，调用后使当前线程进入TIMED_WAITING状态，此状态下会让出CPU的资源并且释放对象的机锁(暂时失去机锁，wait(long timeout)时间达到后返还对象机锁)供其他线程使用
    2. wait()方法是Object类里的方法，使用时必须使用Synchronized包裹，否则会报java.lang.IllegalMonitorStateException异常
    3. 使用其他线程调用同一对象的notify()，notifyAll()或者设置指定睡眠时间来唤醒当前等待池中的线程，使其进入READY状态，等待CUP调度执行

  • 两者的区别

    1. 最明显的区别：sleep()方法是Thread类中的静态方法，用于线程控制；wait()方法是Object类里的方法，用于线程间的通信
    2. 最核心的区别：「是否让出CPU资源」 和「是否释放锁」
      sleep()：让出CPU资源，不释放锁资源。如果sleep方法包裹在Synchronize中，锁并不会释放，因此sleep方法与锁没有直接关系
      wait()：让出CPU资源，也释放锁资源。wait()方法必须放在同步代码块中，执行该方法的线程会释放锁

Java 注解（Annotation）又称 Java 标注，是 JDK5.0 引入的一种注释机制。 注解是元数据的一种形式，提供有关
于程序但不属于程序本身的数据。注解对它们注解的代码的操作没有直接影响

自定义注解

元注解

  • 1.@Documented

    用于Java DOC工具提取成文档

  • 2.@Inherited

    表示允许子类继承父类中定义的注解

  • 3.@Target

    注解标记另一个注解，以限制可以应用注解的 Java 元素类型。目标注解指定以下元素类型之一作为其值：
    ElementType.ANNOTATION_TYPE 可以应用于注解类型
    ElementType.CONSTRUCTOR 可以应用于构造函数
    ElementType.FIELD 可以应用于字段或属性
    ElementType.LOCAL_VARIABLE 可以应用于局部变量
    ElementType.METHOD 可以应用于方法级注解
    ElementType.PACKAGE 可以应用于包声明
    ElementType.PARAMETER 可以应用于方法的参数
    ElementType.TYPE 可以应用于类的任何元素

  • 4.@Retention

    注解指定标记注解的存储方式：
    RetentionPolicy.SOURCE - 标记的注解仅保留在源级别中，并被编译器忽略
    RetentionPolicy.CLASS - 标记的注解在编译时由编译器保留，但 Java 虚拟机(JVM)会忽略
    RetentionPolicy.RUNTIME - 标记的注解由 JVM 保留，因此运行时环境可以使用它
